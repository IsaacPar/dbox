# DrawBoxing

A very small package made to draw boxes with text.

## Usage

```js
import dbox from "drawboxing"

dbox.drawBox(
`
Hey there!

Avaliable on Github ( soon )!
`
)

```

## Known Issues

1. Emojis can move the borders to the left sometimes.
2. ANSI Codes inside main text will break.
